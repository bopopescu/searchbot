To run the front end:
	cd into the <frontend> directory
	make sure bottle is installed, if not the folder bottle-0.12.7 is provided, in command line run:
		cd bottle-0.12.7
		python setup.py install --user
	Then go back into <frontend> directory and run:
		python FindBot.py
	In the browser go to the server bottle has started up.
